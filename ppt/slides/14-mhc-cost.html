This is the classic algorithm for projecting an arbitrary matrix onto the doubly stochastic manifold. Alternating row and column normalization 20 times is enough to make the projection error negligible.

The overhead is real, but compared with the return in deep-model stability, it is still an excellent trade. Without that stability, the run may not finish at all.
mHC is not a performance trick. It is a trainability trick: it turns a 1.6T, 61-layer model from “cannot be trained reliably” into “can be trained stably”.
