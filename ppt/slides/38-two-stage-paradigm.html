V4 TWO-STAGE PARADIGM · OVERVIEW

Divide, then recombine: Specialist → OPD

The report says it plainly: “the mixed RL stage was entirely replaced by On-Policy Distillation.” RL is isolated inside the specialist stage; the student model only performs stable distillation.
The simplest analogy is: first train a set of specialists, then let a generalist learn from watching them. The generalist never runs the hard RL surgery itself. It gains broad capability by aligning to the experts.
