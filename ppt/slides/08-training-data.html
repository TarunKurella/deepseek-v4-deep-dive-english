Chapter 1 · How the 32T/33T data mix was built

More data is not enough. The real question is how to avoid collapse.

33T
V4-Pro pretraining tokens / V4-Flash at 32T
DATA 01
Filter out AI-generated text
The open internet already contains large amounts of ChatGPT- and Claude-written text. DeepSeek explicitly identifies and filters it before training to reduce model collapse risk.
DATA 02
Inject agentic data during mid-training
Tool-use traces, multi-step reasoning, and search fragments are introduced before post-training. Agent behavior is baked into pretraining instead of forced in later.
DATA 03
Expand the multilingual long tail
Lower-resource languages are intentionally topped up so that high-resource languages do not absorb all the capacity while the long tail falls behind.
DATA 04
Curate scientific papers and technical reports
High-quality technical literature is separately up-sampled, directly supporting later performance in scientific reasoning.
Key technical detail: DeepSeek enables sample-level attention masking so that samples packed into the same sequence do not contaminate one another. That is essential infrastructure at 32T-scale training.
