Chapter 0 · Reframing the release
This is not the release of a world-best model.
It is the release that puts a 1M-context Agent model into ordinary users’ hands.
In plain terms: on the frontier-capability race track, Claude Opus 4.7 and GPT-5.5 were already ahead. V4 is not trying to catch them head-on. It is trying to make 1M context, agent tooling, and open weights cheap enough for broad use. That kind of accessibility is rarer than a narrow point lead.
