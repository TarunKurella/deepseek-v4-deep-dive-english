CHAPTER 02 · MHC CORE IDEA

Put a sluice gate on every residual path

"Classic residual connections let every layer’s signal accumulate freely.
mHC constrains those accumulation weights onto a doubly stochastic matrix manifold: every input path sums to 1, and every output path also sums to 1.
INTUITION · IN PLAIN ENGLISH Imagine a water network between layers: each inlet can inject at most 1 unit of water, and each outlet can receive at most 1 unit. No matter how deep the network gets, the total water is conserved. That is what “doubly stochastic” buys you. The guardrail is not soft regularization. It is mathematically enforced.
KEY PROPERTY 01
Spectral norm ≤ 1
A doubly stochastic matrix naturally has spectral norm ≤ 1, so signal magnitude cannot be amplified.
KEY PROPERTY 02
Closed under multiplication
The product of doubly stochastic matrices is still doubly stochastic, so the constraint persists automatically through depth.
KEY PROPERTY 03
Learnable and differentiable
Sinkhorn-Knopp iteration projects an arbitrary matrix onto the manifold, so backprop can optimize it directly.
